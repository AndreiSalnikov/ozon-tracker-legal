Политика обработки персональных данных

Редакция: 2026-02-23 · Версия: v1

1. Оператор и контакты

Оператор: ФИО (самозанятый, НПД), ИНН: XXXXXXXX

Контакт: your-email@example.com

2. Какие данные обрабатываются

  • Идентификаторы Telegram: chat_id, username (если есть), имя (если передано Telegram).
  • Данные для работы сервиса: ссылки на товары, настройки уведомлений, история изменений цен.
  • Данные платежей: идентификатор платежа, сумма, статус, технические метаданные (в т.ч. версия оферты), если подключена оплата.

3. Цели обработки

  • Предоставление функционала бота (мониторинг, уведомления, история).
  • Поддержка пользователей и обработка обращений.
  • Проведение платежей и учёт подписки (для Pro).

4. Правовые основания

Обработка осуществляется для исполнения договора (оферты) и по согласию пользователя в части, необходимой для работы сервиса.

5. Кому могут передаваться данные

  • Telegram — как платформа обмена сообщениями.
  • Ozon — как источник публичной информации о товарах (сервис получает данные с сайта/страниц товаров).
  • ЮKassa — для приёма платежей (если включена оплата).

6. Сроки хранения

Данные хранятся столько, сколько необходимо для работы сервиса и выполнения обязательств, либо до удаления по запросу пользователя, если иное не требуется законом.

7. Безопасность

Оператор применяет разумные меры для защиты данных. Однако пользователь понимает, что передача данных через интернет и Telegram может иметь риски, зависящие от сторонних систем.

8. Права пользователя

Пользователь может запросить удаление данных и/или прекращение обработки, написав на email поддержки. Удаление может быть ограничено обязательствами по хранению информации о платежах и иных данных, если это требуется законом.
